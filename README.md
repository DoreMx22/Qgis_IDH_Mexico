# Qgis_IDH_Mexico
Municipal Human Development Index (HDI) Analysis in Mexico: Resolving Data Gaps

This project features a comprehensive spatial analysis of the Human Development Index (HDI) across Mexico's municipalities. The primary challenge addressed was the integration and standardization of data for the state of Oaxaca—a complex region with 570 municipalities—which previously contained significant information gaps in the national dataset.

Applied Skills & Tools:

Data Processing (Microsoft Excel):

Executed advanced data cleaning and standardization for a dataset of 570 records.

Developed a regional classification logic to assign estimated HDI values to data-void municipalities, ensuring a seamless 100% coverage of the national territory.

Managed geographical keys (CVEGEO) to ensure perfect compatibility with official cartographic shapefiles.

Geospatial Analysis (QGIS):

Performed complex table joins between tabular data and vector layers.

Configured categorized symbology to visualize urban, rural, and transition zones.

Resolved Coordinate Reference System (CRS) conflicts to unify layers from disparate sources into a single, cohesive projection.
